lifeHandlr
==========

Multiplayer god game, in browser played thanks WebGL.

__NOTA__ : *Work In Progress*

Prototype version

REQUIREMENTS
------------

 * node.js
 * Browser with WebGL supported


INSTALL
-------

Type in a terminal :

    $ npm install

RUN
---

For run the server, you can type :

    $ make

The default port is ’3000’, you can test [here](http://localhost:3000).
